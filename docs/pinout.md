# ピン配置 / Pinout

本ページでは、もふもふセンサーボードに搭載されている
RP2040（Raspberry Pi Pico 互換）と、各信号の対応関係を示します。

---

## CH101 センサ接続（基板上 J1 / J2）

| CH101 ピン名 | ピン番号 | RP2040 GPIO | 説明 |
| --- | --- | --- | --- |
| INT | 1 | GP2 | 割り込み出力。測定完了通知などに使用 動作モードによっては入力としても使用される |
| SCL | 2 | GP7 | I2C クロック入力 |
| SDA | 3 | GP6 | I2C データ入出力 |
| PROG | 4 | GP1 | プログラム有効ピン |
| RESET_N | 8 | GP0 | リセット |

---

## Grove コネクタ（UART）

Grove コネクタは **3.3V UART** として使用します。

| Grove Pin | RP2040 GPIO | 機能 |
| --- | --- | --- |
| PIN1 | GP12 | UART_TX |
| PIN2 | GP13 | UART_RX |
| PIN3 | GND | グラウンド |
| PIN4 | 5V0 | 電源入力（5.0V） |

---

## 注意事項

- Grove コネクタは **3.3V ロジックレベル**です。  
  5V ロジックの MCU を接続する場合は、レベル変換を行ってください。
- Grove コネクタ経由で使用する場合は、  
  Grove 向けに I/O 定義を調整した派生ファームウェアを使用してください。
